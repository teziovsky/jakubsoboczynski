## Get started 🏁

Clone the project and go to the project directory

```bash
  git clone git@github.com:teziovsky/jakubsoboczynski.git && cd jakubsoboczynski
```

Install dependencies

```bash
  npm install
```

Start the application!

```bash
  gridsome develop
```
